AI Explainer Reels Factory

An end-to-end pipeline that takes a topic prompt and returns a finished short-form educational video. The system chains LLM script planning, TTS voiceover synthesis, automated asset generation, subtitle burning, and Remotion video rendering into a single workflow. Exposed over FastAPI with a Streamlit control UI.

The design goal was a real workflow, not a demo. Given a prompt, it handles every layer from script to final MP4 without human intervention.
